This repository contains **carefully crafted prompts** designed to generate **high-quality illustrative / cinematic images** using a **user-uploaded photo as the primary facial reference**.

The goal of these prompts is to:
- Preserve **exact facial identity**
- Avoid face beautification or distortion
- Generate **photorealistic fantasy, cinematic, or artistic visuals**
- Maintain realism while adding **stylized elements** (energy effects, outfits, environments, etc.)

Sample outputs generated using these prompts are included in the **Assets** folder.
