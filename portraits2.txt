# Portraits
# ==========

"Create 9 ultra-realistic 8K portraits of the same man using the 
uploaded photo exactly (100% identical facial identity). 
Arrange them in a 3x3 grid (portrait ratio) with each image 
showing him in a different color suit and tie (mix of solid and checked). 
Each scene should feature a unique pose, gesture, and background color tone."
Add subtle typography watermark at the bottom-center 
reading: “Created By : ALOK KUMAR KAUSHAL” — small but readable, 
neon orange glowing text with a soft outer glow. Apply an MS-Office-style outer border stroke 
using a smooth Blue → Green gradient, luminous, clean, modern, not overpowering the artwork.
